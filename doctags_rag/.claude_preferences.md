# Claude Communication Preferences

## Vocabulary Restrictions

**NEVER use these words in conversation:**
- "absolutely"
- "excellent"
- "production"

## Rationale
User preference - these terms are overused and should be avoided in all responses.

## Alternative Expressions

Instead of "absolutely":
- "yes"
- "confirmed"
- "understood"
- "correct"

Instead of "excellent":
- "good"
- "strong"
- "solid"
- "impressive"
- "outstanding"

Instead of "production":
- "deployment"
- "live environment"
- "operational use"
- "real-world use"
