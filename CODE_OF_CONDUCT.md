# Code Of Conduct

## Our standard

We are committed to a respectful, inclusive, and constructive project community.

Expected behavior:

1. Be respectful in language and tone.
2. Focus feedback on technical issues and outcomes.
3. Assume good intent and ask for clarification when needed.
4. Avoid harassment, discrimination, and personal attacks.

## Reporting issues

Please report conduct issues privately through repository maintainers.

## Enforcement

Project maintainers may remove comments, close discussions, or block access for violations.
