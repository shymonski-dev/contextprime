# Support

Use these channels for help:

1. Open an issue for bug reports and feature requests.
2. Open a discussion for usage questions.
3. Check `README.md`, `QUICK_REFERENCE.md`, and `SECURITY.md` first.

When reporting issues, include:

1. Exact command used.
2. Relevant logs and error output.
3. Environment details (operating system, container version, Python version).
